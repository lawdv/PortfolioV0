Todo List con SQL Server como base de datos

Aplicación web para gestionar tareas, con un backend en SQL Server para almacenar y organizar listas de tareas de forma eficiente.

Frontend: Desarrollado con React 19 y Next.js. Se usó Tailwind CSS para estilos y Socket.IO para comunicación en tiempo real. Incluye íconos con FontAwesome. Backend: Construido con Node.js y Express. Conexión a base de datos MSSQL, uso de dotenv para variables de entorno, y comunicación en tiempo real con Socket.IO
